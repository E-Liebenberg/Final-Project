Welcome to the Nurse Call System

This platform connects bedside units, patient remotes, and theatre modules into a single real-time monitoring system. Nurses receive instant alerts for calls, Code Blue emergencies, and sensor events (movement, sound, RFID, temperature, humidity).

Use the navigation bar above to admit a patient, review pending admissions, or open a live ward or theatre dashboard. Each action is logged for safety, ensuring accountability in critical hospital environments.
